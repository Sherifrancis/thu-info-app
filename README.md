## Build

### Requirements

All that you need to build a [React Native](https://reactnative.cn/docs/getting-started.html) project.

### Procedure

```bash
git clone https://github.com/UNIDY2002/THUInfo.git
cd THUInfo
yarn
yarn android  # For Android
yarn ios      # For iOS
```
